# 腾讯云 · React Native 在线直播解决方案

<img src="https://qcloudimg.tencent-cloud.cn/raw/ec034fc6e4cf42cae579d32f5ab434a1.png" align="left" width=65 height=65>

TUILiveKit 是一款适用于社交娱乐、购物、健身等互动直播场景的产品，通过集成该产品，仅需三步，30分钟内就可以为您的 App 添加互动连麦、送礼、房间管理等功能。

## 产品特性

<p align="center">
  <img src="https://web.sdk.qcloud.com/component/uni-app/assets/clipboard_20250903_054158.png"/>
</p>

- **完善的 UI 组件**：我们提供开箱即用的直播页面组件，包括主播开播、观众观看、直播列表等完整页面，可以节省您 90% 的开发时间，您只需要花费30分钟就可以拥有一款完整的在线直播应用。
- **跨平台支持**：我们支持 Android、iOS 双平台，确保您在不同平台上的体验一致性。
- **互动弹幕**：我们支持自定义文本，自定义表情，房间信息提醒等多种互动信息展示。
- **互动礼物**：我们支持自定义礼物动画展示，费用自定义结算等功能。
