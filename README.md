# Tencent Cloud · React Native Live Streaming Solution

<img src="https://qcloudimg.tencent-cloud.cn/raw/ec034fc6e4cf42cae579d32f5ab434a1.png" align="left" width=65 height=65>

TUILiveKit is a product suitable for interactive live streaming scenarios such as social entertainment, shopping, and fitness. By integrating this product, you can add interactive co-hosting, gifting, and room management features to your App in just three steps within 30 minutes.

## Product Features

<p align="center">
  <img src="https://web.sdk.qcloud.com/component/uni-app/assets/clipboard_20250903_054158.png"/>
</p>

- **Comprehensive UI Components**: We provide ready-to-use live streaming page components, including anchor broadcasting, audience viewing, and live list pages. This can save you 90% of development time. You only need 30 minutes to have a complete online live streaming application.
- **Cross-platform Support**: We support Android and iOS platforms, ensuring a consistent experience across different platforms.
- **Interactive Bullet Comments**: We support custom text, custom emojis, room information reminders, and other interactive information display.
- **Interactive Gifts**: We support custom gift animation display, customizable payment settlement, and other features.
